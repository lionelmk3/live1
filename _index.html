Mon premier html CONCEPT GIT 3 parties source index depot fonctions : git init (initialise un repertoire git) git status (check le dossier et le status) git add filename (ajouter des fichiers à l’index) git add -A (tout ajouter à l'index) git rm --cached filename (supprime un fichier de l'index) git commit -m "objet" (passer de l'index au dépot) git log (log d’une branche) git log -1 (dernier log de la branche) git log --oneline (une seule ligne pour les logs) git shortlog (seulement les objets des commit) git checkout branchname (bascule vers la branche) git checkout -b branchname (création d'un branche)
